Encrypt Tools – Juned KW

Encrypt Tools adalah aplikasi berbasis CLI (terminal) yang menyediakan beberapa metode encoding/encryption sederhana untuk kebutuhan proteksi kode/file, pembelajaran, dan obfuscation ringan.

> ⚠️ Peringatan Tools ini tidak ditujukan untuk aktivitas berbahaya/malicious. Gunakan hanya untuk tujuan edukasi, pengamanan pribadi, atau obfuscation legal.

# Fitur Utama:

1. Base64 + zlib - Enkripsi Python dengan kompresi
2. Marshal - Enkripsi bytecode Python
3. Decrypt Base64 - Dekripsi metode [01]
4. Marshal Include - Enkripsi Python dengan dependensi
5. JavaScript Encrypt - Obfuskasi kode JS
6. Bash Encrypt - Enkripsi skrip shell
7. XOR Encrypt - Enkripsi sederhana Python
8. HTML Encode - Konversi karakter khusus HTML

# Catatan:

· Hanya untuk tujuan edukasi & keamanan sah
· Patuhi hukum yang berlaku
· Backup data sebelum enkripsi
