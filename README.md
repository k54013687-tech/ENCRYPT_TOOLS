Encrypt Tools – Juned KW

Encrypt Tools adalah aplikasi berbasis CLI (terminal) yang menyediakan beberapa metode encoding/encryption sederhana untuk kebutuhan proteksi kode/file, pembelajaran, dan obfuscation ringan.

> ⚠️ Peringatan Tools ini tidak ditujukan untuk aktivitas berbahaya/malicious. Gunakan hanya untuk tujuan edukasi, pengamanan pribadi, atau obfuscation legal.


#### Tested On  :

* Kali Linux

* BlackArch Linux

* Kali Nethunter

* Termux ( Rooted/NonRooted Devices)
